# lissajous

将格式为 jpg, png, jpeg, gif 的图片转为 jpeg 格式的工具

## Abstract

Lissajous 帮助您将 jpg, png, gif 格式的图片转为更小的 jpeg 图片

##  Usage

| 参数 | 类型   | 说明                            | 默认值      |
| ---- | ------ | ------------------------------- | ----------- |
| -d   | string | 指定需要转换的文件名称或文件夹  | .           |
| -e   |        | 查看当前运行环境                |             |
| -h   |        | 查看使用方法                    |             |
| -o   | string | 输出到文件夹                    | ._converted |
| -q   | int    | 输出的图片质量为原图片的 1~100% | 75          |
| -v   |        | 查看当前版本                    |             |
